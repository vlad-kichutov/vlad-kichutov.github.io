1.9 Условные операторы if, else, switch

Условные операторы позволяют программе принимать решения — выполнять разные блоки кода в зависимости от условий. Это основа логики в программировании. Сегодня ты узнаешь про if, else и switch.

🤔 Что такое условие?​

Условие — это выражение, которое возвращает true или false. Например:

int age = 18;
boolean isAdult = age >= 18; // true

На основе такого выражения программа может решить, какой путь выбрать.

✅ Оператор if​

if (условие) {
    // код выполнится, если условие истинно
}

Пример:

int age = 20;
if (age >= 18) {
    System.out.println("Доступ разрешён");
}

Если age >= 18 — условие true, код выполняется. Если нет — блок пропускается.

✅ if + else​

if (условие) {
    // если условие true
} else {
    // если условие false
}

Пример:

int age = 16;
if (age >= 18) {
    System.out.println("Доступ разрешён");
} else {
    System.out.println("Доступ запрещён");
}

✅ Цепочка if - else if - else​

if (условие1) {
    // блок 1
} else if (условие2) {
    // блок 2
} else {
    // блок 3, если ни одно условие не подошло
}

Пример:

int temp = 5;
if (temp > 20) {
    System.out.println("Тепло");
} else if (temp > 0) {
    System.out.println("Прохладно");
} else {
    System.out.println("Холодно");
}

🔁 Оператор switch​

Используется, когда нужно сравнить одно значение с несколькими вариантами:

switch (переменная) {
    case значение1:
        // код
        break;
    case значение2:
        // код
        break;
    default:
        // если ничего не подошло
}

Пример:

int day = 3;
switch (day) {
    case 1:
        System.out.println("Понедельник");
        break;
    case 2:
        System.out.println("Вторник");
        break;
    case 3:
        System.out.println("Среда");
        break;
    default:
        System.out.println("Другой день");
}

Важно: break завершает выполнение. Без него Java будет продолжать выполнять следующий блок (fall-through).

🔍 Когда использовать if, а когда switch?​

if-elseswitch
Сложные условия (>, <, &&)Много простых вариантов одного значения
Несколько условийПроверка значения одной переменной
Можно комбинировать условияТолько сравнение на равенство

🧪 Задания для самопроверки​

  1. Напиши программу, которая проверяет, положительное ли число, отрицательное или ноль.
  2. Используя if-else, выведи сообщение о времени суток: утро, день, вечер, ночь — по значению переменной hour (от 0 до 23).
  3. С помощью switch напиши программу, которая по числу от 1 до 7 выводит день недели.
  4. Убери break в switch — что произойдёт? Проверь сам!
