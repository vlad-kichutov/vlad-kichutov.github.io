1.13 Мини-проект: Угадай число

Давай закрепим всё, что мы выучили в модуле, с помощью еще одного мини-проекта! Мы реализуем игру, в которой компьютер загадывает число, а пользователь должен его угадать. Это отличная практика для if, while, break, continue и ввода данных с клавиатуры.

🎯 Цель проекта​

Создать программу, которая:

  • Случайным образом загадывает число от 1 до 100
  • Запрашивает число у пользователя и выдает подсказку: "больше" или "меньше"
  • Продолжается до тех пор, пока пользователь не угадает число
  • После угадывания — сообщается количество попыток

🧠 План действий​

Попробуй продумать его самостоятельно

🎲 Как работает Random?​

Для генерации случайного числа мы используем класс Random:

Random random = new Random();
int number = random.nextInt(100) + 1; // от 1 до 100
  • random.nextInt(100) генерирует число от 0 до 99
  • +1 сдвигает диапазон до 1–100

Это ключевой момент, который делает игру интересной и каждый раз уникальной!

💻 Код программы​

осторожно

Открывай только если ты уже выполнил задание самостоятельно!

примечание

Код программы может отличаться от твоего решения

Показать/скрыть код
import java.util.Scanner;
import java.util.Random;

public class GuessTheNumber {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Random random = new Random();

        int secretNumber = random.nextInt(100) + 1; // от 1 до 100
        int attempts = 0;
        int guess;

        System.out.println("Угадай число от 1 до 100!");

        while (true) {
            System.out.print("Твой вариант: ");
            guess = scanner.nextInt();
            attempts++;

            if (guess < secretNumber) {
                System.out.println("Загаданное число больше");
            } else if (guess > secretNumber) {
                System.out.println("Загаданное число меньше");
            } else {
                System.out.println("Поздравляю! Ты угадал с " + attempts + " попытки!");
                break; // выходим из цикла
            }
        }

        scanner.close();
    }
}

🔍 Пояснение к коду​

  • Используем Scanner для ввода и Random для генерации числа.
  • Цикл while (true) крутится до угадывания.
  • Сравниваем введённое число с загаданным.
  • Подсказываем пользователю: больше/меньше.
  • break завершает игру, когда число угадано.

🧪 Что можно улучшить?​

  1. Проверка границ: предупредить, если пользователь ввёл число не от 1 до 100.
  2. Попробуй добавить лимит попыток. Например, 7 попыток, а потом — поражение.
  3. Предложить сыграть ещё раз после окончания игры (используй do-while).
