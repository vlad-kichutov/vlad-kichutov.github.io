1.2 Установка IntelliJ IDEA и создание первого проекта 🚀

Прежде чем писать код, нужно подготовить рабочее место. В этом уроке ты узнаешь, как установить одну из самых популярных сред разработки — IntelliJ IDEA от компании JetBrains, и как создать свой первый проект на Java.

🛠 Шаг 1: Установка IntelliJ IDEA​

1. Перейди на сайт JetBrains:​

👉 https://www.jetbrains.com/idea/download/

2. Выбери версию:​

  • Community Edition — бесплатная. Подходит для обучения и большинства задач.
  • Ultimate Edition — платная, нужна в основном для профессиональной разработки веб-приложений.

👉 Нам подойдёт Community Edition. Нажми Download и скачай установщик.

3. Установи IDE:​

  • Запусти установщик и следуй инструкциям.
  • На этапе выбора компонентов можешь оставить всё по умолчанию.
  • После установки — запусти IntelliJ IDEA.

🧰 Шаг 2: Установка JDK и создание нового проекта​

После запуска IntelliJ IDEA ты увидишь следующее окно. Выбери New Project.

  • В левом меню выбери язык программирования Java.
  • В поле Name введи название нового проекта, например java-course.
  • В графе Build system выбери IntelliJ.
  • Скорее всего в графе JDK у тебя красным цветом будет подсвечено <No JDK>. Кликни по этой надписи и раскрой выпадающий список.
  • В выпадающем списке выбери пункт Download JDK....
  • В поле Version выбери 17.
  • В поле Vendor выбери Amazon Corretto.
  • Поле Location оставь без изменений.
  • По итогу твое окно создания проекта должно выглядеть следующим образом.
  • Нажми на кнопку Create.

👨‍💻 Шаг 4: Создание первого Java-файла​

  • Перед тобой откроется следующее окно.
  • В левой панели (Project) щёлкни правой кнопкой по папке src
  • Выбери: New → Java Class
  • Назови класс, например: Main
  • В папке src у тебя появится файл с названием Main.java
  • А внутри файла будем немного кода

Молодец! Ты установил IntelliJ IDEA, подключил JDK, создал проект и первый Java файл. Теперь ты готов двигаться дальше и написать свою первую программу! 💪
