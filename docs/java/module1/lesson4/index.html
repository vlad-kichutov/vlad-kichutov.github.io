1.4 Примитивные типы данных и переменные в Java

Java — строго типизированный язык. Это значит, что каждая переменная должна иметь определённый тип данных, и этот тип нельзя просто так поменять в процессе. Давай разберёмся, какие бывают типы, зачем они нужны и как объявлять переменные.

📦 Что такое переменная?​

Переменная — это именованная ячейка в памяти, в которой хранится какое-то значение. У каждой переменной есть имя и тип.

Переменная age типа int со значением 25

int age = 25;

🔢 Примитивные типы данных​

Java предлагает 8 примитивных типов данных:

ТипОписаниеПример значенияДиапазон значений
byteцелое число (1 байт)127от -128 до 127
shortцелое число (2 байта)32000от -32,768 до 32,767
intцелое число (4 байта)100000от -2,147,483,648 до 2,147,483,647
longбольшое целое (8 байт)10000000000Lот -9,223,372,036,854,775,808 до 9,223,372,036,854,775,807
floatдробное число (4 байта)3.14f~ ±3.4×10^38 (7 знаков точности)
doubleдробное число (8 байт, точнее)3.141592~ ±1.8×10^308 (15 знаков точности)
charсимвол Unicode (2 байта)'A'от \u0000 до \uffff (0–65535)
booleanлогическое значениеtrue / falseтолько true или false

Примеры:​

int number = 42;
long bigNumber = 10000000000L;
boolean isJavaFun = true;
char grade = 'A';
float price = 10.5f;
double pi = 3.14;

Обрати внимание: для float нужно писать f в конце, для long — L

✍️ Как объявлять переменные​

Сначала указываем тип, потом имя переменной, затем (опционально) — значение:

int score;
score = 100;

или сразу:

int score = 100;

🧠 Именование переменных​

  • Используй понятные имена (age, userName, isLoggedIn)
  • Имя должно начинаться с буквы, может содержать цифры и _, но не начинаться с цифры
  • Нельзя использовать зарезервированные слова (int, class, public, и т.д.)

🧪 Задания для самопроверки​

  1. Создай переменные всех 8 типов и присвой им значения.
  2. Объяви переменную float и попробуй не добавить f в конце. Что произойдёт?
  3. Придумай 3 хороших и 3 плохих имени переменных. Почему плохие — плохие?

Теперь ты умеешь создавать переменные и выбирать для них подходящие типы. Отличная работа! 🚀
