1.11 Класс Scanner (ввод данных в Java)

В большинстве программ нужно как-то взаимодействовать с пользователем: спросить имя, запросить число, получить команду. В Java для этого чаще всего используется класс Scanner.

🔍 Что такое Scanner​

Scanner — это класс из пакета java.util, который позволяет считывать ввод от пользователя с клавиатуры (или из других источников).

Перед началом работы с ним нужно подключить импорт:

import java.util.Scanner;

🔤 Пример: считывание строки​

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); // создаём сканнер

        System.out.print("Введите ваше имя: ");
        String name = scanner.nextLine(); // читаем строку

        System.out.println("Привет, " + name + "!");
    }
}
  • System.in — стандартный поток ввода (клавиатура)
  • nextLine() — считывает целую строку до нажатия Enter

🔢 Считывание чисел​

  • nextInt() — целое число
  • nextDouble() — дробное число
  • next() — одно слово
System.out.print("Введите возраст: ");
int age = scanner.nextInt();

System.out.print("Введите рост: ");
double height = scanner.nextDouble();

⚠️ Важно: если сначала считать число, а потом строку — может возникнуть проблема с переходом на следующую строку. В таких случаях используй дополнительный scanner.nextLine();, чтобы "очистить" переход.

❌ Возможные ошибки​

Если пользователь введёт не то, что ожидается (например, текст вместо числа), программа завершится с ошибкой. Этого можно избежать, проверяя ввод заранее или обрабатывая исключения (мы позже это изучим).

🧪 Задания для самопроверки​

  1. Запроси у пользователя его имя и возраст. Выведи фразу: "Имя: ___, Возраст: ___"
  2. Спроси у пользователя два числа и выведи их сумму.
  3. Запроси строку и число. Что произойдёт, если поменять порядок?
  4. Придумай мини-опросник: имя, любимый цвет, любимое число. Выведи красиво оформленный результат.

Теперь ты умеешь делать программы, которые взаимодействуют с пользователем. Это уже похоже на настоящее приложение! 😉
