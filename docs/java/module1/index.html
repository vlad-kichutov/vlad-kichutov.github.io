Привет! 👋 Если ты читаешь это — значит, ты хочешь научиться программировать.

📄️ 1.2 Установка IDEA и создание нового проекта

Прежде чем писать код, нужно подготовить рабочее место. В этом уроке ты узнаешь, как установить одну из самых популярных сред разработки — IntelliJ IDEA от компании JetBrains, и как создать свой первый проект на Java.

📄️ 1.3 Первая программа Hello world

Теперь, когда ты установил IntelliJ IDEA и создал проект — пора написать первую настоящую программу на Java. Это будет классическое "Hello, World!" — фраза, с которой начинают все программисты в мире 🌍

📄️ 1.5 Комментарии

Когда ты пишешь код, важно не только что ты пишешь, но и как понятно это будет другим (и тебе самому через месяц). Комментарии — это специальные пояснения внутри кода, которые игнорируются при выполнении, но помогают понять, что происходит.

📄️ 1.6 Зарезервированные слова

В любом языке программирования есть специальные слова, которые имеют особое значение. Такие слова называются зарезервированными или ключевыми словами.

📄️ 1.7 Операции над переменными

Теперь, когда ты знаешь, как объявлять переменные и какие типы данных бывают — пора научиться взаимодействовать с ними. В этом уроке мы рассмотрим арифметику, логические операции, операторы присваивания и многое другое.

📄️ 1.8 Строки String и конкатенация

В этом уроке ты познакомишься со строками в Java — типом данных для хранения текста. Мы разберём, как создавать строки, объединять их (это называется конкатенация) и использовать в выражениях.

📄️ 1.9 Условные операторы if, else, switch

Условные операторы позволяют программе принимать решения — выполнять разные блоки кода в зависимости от условий.

📄️ 1.10 Циклы while, do-while, for и for-each

Циклы позволяют выполнять одни и те же действия многократно, пока выполняется определённое условие. Это один из самых мощных инструментов в программировании — и ты сейчас научишься им пользоваться.

📄️ 1.11 Класс Scanner (ввод данных)

В большинстве программ нужно как-то взаимодействовать с пользователем: спросить имя, запросить число, получить команду. В Java для этого чаще всего используется класс Scanner.
