1.8 Строки и конкатенация в Java

В этом уроке ты познакомишься со строками в Java — типом данных для хранения текста. Мы разберём, как создавать строки, объединять их (это называется конкатенация) и использовать в выражениях.

✍️ Что такое String​

String — это не примитивный тип, а объект, предназначенный для хранения текста.

String name = "Анна";
System.out.println(name);

Строки в Java записываются в двойных кавычках: "текст"

➕ Конкатенация строк​

Конкатенация — это объединение строк. В Java это делается с помощью оператора +.

String firstName = "Иван";
String lastName = "Иванов";
String fullName = firstName + " " + lastName;
System.out.println(fullName); // Иван Иванов

Можно объединять строки с числами и другими типами:

int age = 30;
System.out.println("Возраст: " + age); // Возраст: 30

Java автоматически преобразует int, double, boolean и другие типы в строку при конкатенации.

📐 Длина строки​

У каждого объекта String есть метод .length(), который возвращает количество символов:

String word = "Привет";
System.out.println(word.length()); // 6

🔢 Доступ к символам строки​

Символы внутри строки можно получать с помощью метода .charAt(index):

String text = "Java";
char first = text.charAt(0); // 'J'
System.out.println(first);

Индексация начинается с 0!

🔄 Преобразование строки и числа​

Иногда нужно превратить число в строку и наоборот:

Число → строка:

int num = 10;
String result = String.valueOf(num);

Строка → число:

String str = "42";
int parsed = Integer.parseInt(str);

📛 Экранирование символов​

Некоторые символы нельзя писать напрямую — для этого используют экранирование с помощью \:

System.out.println("Он сказал: \"Привет!\"");

Полезные последовательности:

  • \n — перенос строки
  • \t — табуляция
  • \\ — обратный слэш

🧪 Задания для самопроверки​

  1. Объяви строку с твоим именем и фамилией. Объедини их с пробелом и выведи результат.
  2. Используй .length(), чтобы узнать длину своего имени.
  3. Получи 3-й символ строки "Программирование" с помощью .charAt()
  4. Преобразуй строку "100" в int и прибавь к ней 50. Выведи результат.
  5. Напиши строку с кавычками внутри: Она сказала: "Я люблю Java!"

Теперь ты умеешь работать с текстом в Java, объединять строки, извлекать из них символы и комбинировать с другими типами. Строки — это огромная и важная тема, и мы ещё к ней вернёмся! 😉
