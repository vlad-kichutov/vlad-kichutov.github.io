1.7 Операции над переменными в Java

Теперь, когда ты знаешь, как объявлять переменные и какие типы данных бывают — пора научиться взаимодействовать с ними. В этом уроке мы рассмотрим арифметику, логические операции, операторы присваивания и многое другое.

➕ Арифметические операции​

Арифметические операторы работают с числами (int, float, double и др.):

ОператорОписаниеПримерРезультат
+Сложение5 + 38
-Вычитание5 - 32
*Умножение5 * 315
/Деление10 / 25
%Остаток от деления10 % 31

Пример:

int a = 10;
int b = 3;
System.out.println(a + b); // 13
System.out.println(a % b); // 1

⚠️ Деление двух int всегда даёт int. Например: 5 / 2 = 2 (остаток отбрасывается).

➕ Операции с разными типами​

Java может автоматически преобразовывать типы:

int a = 5;
double b = 2.5;
double result = a + b; // a преобразуется в double

Результат будет 7.5, потому что int + double → double

🔁 Операторы присваивания​

Можно присваивать значения и комбинировать с другими операциями:

int x = 10;
x = x + 5; // x стал 15

Есть сокращённые формы:

ОператорПримерТо же самое что и
+=x += 5x = x + 5
-=x -= 3x = x - 3
*=x *= 2x = x * 2
/=x /= 4x = x / 4
%=x %= 2x = x % 2

🔢 Инкремент и декремент​

ОператорОписаниеПримерРезультат
++Увеличить на 1x++x = x + 1
--Уменьшить на 1x--x = x - 1
int count = 0;
count++; // теперь count = 1
count--; // снова 0

Можно использовать ++x и x++ — разница в порядке выполнения:

int x = 5;
System.out.println(x++); // сначала выведет 5, потом x станет 6
System.out.println(++x); // x уже стал 7, выведет 7

🧮 Операции сравнения​

Сравнение позволяет сравнивать значения переменных. Результатом всегда будет true или false (тип boolean).

ОператорОписаниеПримерРезультат
==Равно5 == 5true
!=Не равно5 != 3true
>Больше7 > 3true
<Меньше2 < 1false
>=Больше или равно5 >= 5true
<=Меньше или равно3 <= 4true

Пример:

int a = 10;
int b = 5;
System.out.println(a > b); // true
System.out.println(a == 10); // true
System.out.println(b != 0); // true

Такие выражения часто используются в условиях (if, while, и др.).

🔍 Практика с переменными​

int apples = 10;
apples += 5; // теперь 15
apples--;    // теперь 14
System.out.println(apples);

🧪 Задания для самопроверки​

  1. Создай переменную score = 50, увеличь её на 20, потом уменьши на 10 и выведи результат.
  2. Объясни разницу между x++ и ++x на примере.
  3. Какой результат будет у выражения 5 + 3 * 2? Почему?
  4. Что выведет код:
int a = 8;
a %= 3;
System.out.println(a);

Теперь ты умеешь управлять значениями в переменных и делать с ними всё, что угодно. Погнали дальше! 🚀
